# 에피소드 3. 시간의 이차방정식 톱니바퀴 탑

## 3. [학생용 미션 지문]

### 🔮 제 1막: 방정식의 판별 (이차방정식의 정의와 해)
- **Q1.** 다음 보기 중 이차방정식인 것의 번호를 쓰시오.
(1) 2x - 5 = 0
(2) x² - 3x + 2 = 0
(3) x³ + x² - 1 = 0
(4) x² - (x² - 2x) = 0
- **Q2.** 이차방정식 x² - 4 = 0 의 두 근을 구하시오. (± 기호를 사용하여 구하시오. 예: ±2)
- **Q3.** x=2 가 이차방정식 x² - ax + 6 = 0 의 한 근일 때, 상수 a의 값을 구하시오.
- **Q4.** 이차방정식 (x-3)(x+2) = 0 의 두 근을 구하시오. (작은 수부터 쉼표로 구분하여 적으시오. 예: -2, 3)
- **Q5.** 이차방정식 x² - 5x + 6 = 0 의 두 근을 구하시오. (작은 수부터 적으시오. 예: 2, 3)

### 📜 제 2막: 제곱의 힘 (중근과 완전제곱식의 활용)
- **Q6.** 이차방정식 x² - 8x + 16 = 0 의 근을 구하시오.
- **Q7.** 이차방정식 x² + ax + 9 = 0 이 중근을 가질 때, 양수 a의 값을 구하시오.
- **Q8.** 이차방정식 (x-2)² = 5 의 근을 구하시오. (기호 ±와 루트를 사용하여 나타내시오. 예: 2±√5)
- **Q9.** 이차방정식 x² - 6x + 2 = 0 을 (x-p)² = q 꼴로 고칠 때, 두 상수 p, q에 대하여 p+q의 값을 구하시오.
- **Q10.** 근의 공식을 사용하여 이차방정식 x² - 3x + 1 = 0 의 근을 구할 때, 분모에 들어갈 값을 구하시오.

### 🪄 제 3막: 근의 열쇠 (근의 공식과 풀이)
- **Q11.** 이차방정식 x² - 4x - 1 = 0 의 근을 구하시오. (근의 공식을 활용하여 나타내시오. 예: 2±√5)
- **Q12.** 이차방정식 2x² + 5x - 3 = 0 의 두 근을 구하시오. (작은 수부터 적으시오. 분수는 슬래시로 표현하시오. 예: -3, 1/2)
- **Q13.** 이차방정식 (x-1)(x-2) = 6 의 두 근을 구하시오. (작은 수부터 적으시오. 예: -1, 4)
- **Q14.** 두 근이 -2와 5 이고 x²의 계수가 1인 이차방정식을 구하시오. (예: x²-3x-10=0)
- **Q15.** 이차방정식 x² + 4x - 5 = 0 의 두 근 중 양수인 근을 구하시오.

### 🎇 제 4막: 톱니바퀴 재가동 (이차방정식의 활용)
- **Q16.** 이차방정식 x² - 2x = 0 의 두 근을 구하시오. (작은 수부터 적으시오. 예: 0, 2)
- **Q17.** 이차방정식 3x² - 7x + 2 = 0 의 두 근 중 정수인 근을 구하시오.
- **Q18.** 이차방정식 x² - 6x + 9 = 0 의 중근을 구하시오.
- **Q19.** 연속하는 두 자연수의 곱이 56일 때, 두 자연수 중 큰 수를 구하시오.
- **Q20.** 지면에서 초속 30m로 똑바로 쏘아 올린 물체의 t초 후의 높이가 30t - 5t² (m)이다. 이 물체의 높이가 40m가 되는 시점은 몇 초 후인지 모두 구하시오. (오름차순으로 적으시오. 예: 2, 4)

## 4. [교사용 진행 가이드 및 힌트]
- **Q1 힌트:** 주어진 등식을 모두 한 변(좌변)으로 이항하여 정리했을 때, (x에 대한 2차식) = 0 꼴이 되는 식을 찾게 합니다.
- **Q2 힌트:** 미지수의 제곱이 상수항과 같음을 이용해, 제곱근의 정의를 활용해 해를 구하도록 안내합니다.
- **Q3 힌트:** 이차방정식의 한 근이 주어지면, 그 근의 값을 방정식의 미지수 자리에 대입하여 성립함을 활용하게 지도합니다.
- **Q4 힌트:** 두 일차식의 곱이 0이 되기 위해서는 각각의 일차식이 0이 되어야 한다는 방정식을 세우게 합니다.
- **Q5 힌트:** 우변을 0으로 만든 뒤 좌변을 인수분해하여 두 일차식의 곱이 0이 되는 원리로 해를 구하게 유도합니다.
- **Q6 힌트:** 좌변이 동일한 두 일차식의 곱(완전제곱식)으로 인수분해될 때 발생하는 중복된 근(중근)의 성질을 설명합니다.
- **Q7 힌트:** 이차방정식이 중근을 가질 조건은 상수항이 1차항 계수의 반의 제곱과 같아야 한다는 공식을 적용하게 합니다.
- **Q8 힌트:** 완전제곱식을 이용해 제곱근의 정의를 활용하여 미지수 값의 범위를 구하도록 지도합니다.
- **Q9 힌트:** 상수항을 먼저 우변으로 이항한 후, 좌변을 완전제곱식으로 만들기 위해 양변에 필요한 상수를 더해주는 과정을 안내합니다.
- **Q10 힌트:** 이차방정식의 근의 공식 분모 부분을 구하는 문제로, 분모는 x² 계수의 2배가 됨을 상기시킵니다.
- **Q11 힌트:** 인수분해가 되지 않는 경우 근의 공식을 사용하여 해를 대입하여 계산하게 가이드합니다.
- **Q12 힌트:** 대각선 곱셈 인수분해 기법을 활용해 좌변을 두 일차식의 곱으로 유도한 후 해를 구하게 합니다.
- **Q13 힌트:** 괄호 식을 먼저 모두 전개한 후 모든 항을 한 변으로 이항해 정리하여 이차방정식 기본 형태로 바꾸어 풀게 합니다.
- **Q14 힌트:** 두 근이 주어졌을 때 이차방정식의 작성 공식인 (x-a)(x-b) = 0 을 이용하여 전개하도록 지도합니다.
- **Q15 힌트:** 방정식의 해를 구한 후 얻어진 근 중에서 조건에 부합하는 양수 값을 선택하게 안내합니다.
- **Q16 힌트:** 상수항이 0인 경우, 공통인수 x를 묶어내어 간단히 인수분해하여 해를 구하게 유도합니다.
- **Q17 힌트:** 인수분해를 통해 근을 도출하고, 그 중에서 정수 형태를 가진 근만 선별하도록 가이드합니다.
- **Q18 힌트:** 좌변이 완전제곱식이 되어 한 개의 중근을 가지게 되는 식을 판별하게 유도합니다.
- **Q19 힌트:** 두 연속하는 자연수를 x와 x+1 로 두고 곱을 나타내는 방정식을 세워 이차방정식으로 해결하게 유도합니다.
- **Q20 힌트:** 높이 식과 목표 수치가 같다는 방정식을 세운 뒤 최고차항의 계수를 간단히 정리하여 해를 구하도록 안내합니다.
