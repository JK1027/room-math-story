# 방탈출 앱 고도화 아이디어 노트

본 문서는 추후 방탈출 앱의 학생 몰입도를 더욱 끌어올리기 위해 보류(Keep)해둔 기획 아이디어를 정리한 노트입니다.

### 1. 빌런(Antagonist)의 직접적인 도발 시스템
*   **개요**: 조력자 캐릭터가 상황을 중계하는 것을 넘어, 플레이어를 억압하거나 비웃는 메인 빌런 캐릭터가 통신에 난입하여 직접적으로 도발하는 연출.
*   **구현 아이디어**: 특정 문제 진입 시 빌런의 해킹 연출 텍스트 출력. "이 정도 방정식도 못 푸나?" 등의 자극적인 대사 배치.

### 2. 중간 국면 전환 (Plot Twist / 컷씬)
*   **개요**: 20문제가 동일한 전개로 이어지지 않도록, 특정 문항(예: 10번, 15번) 진입 시 갑작스러운 위기 상황을 알리는 컷씬 발생.
*   **구현 아이디어**: "메인 전력 차단! 산소 공급 중단!"과 같은 극적인 상황 변동을 주어, 후반부 문제 풀이의 긴장감을 극대화.

### 3. 조력자 캐릭터의 감정 변화 (Character Arc)
*   **개요**: 기계적이거나 차갑던 조력자 캐릭터가 문제 풀이 진행에 따라 플레이어와 유대감을 형성하고 점점 절박하거나 감동적인 말투로 변화.
*   **구현 아이디어**: 초반에는 무미건조한 시스템 텍스트를 쓰다가 후반부(18~20번)에는 플레이어의 생존을 간절히 바라는 텍스트 출력.

### 4. 멀티 엔딩 시스템 (Multi-Ending System)
*   **개요**: 학생의 플레이 기록(오답 횟수, 클리어 시간 등)에 따라 결말 스토리가 분기되는 시스템.
*   **구현 아이디어**:
    *   **True Ending (완벽한 클리어)**: 오답 횟수가 0회이거나 매우 적을 경우. 
        *   *(예시)* "오라클-X: 경이로운 연산 속도입니다. 시스템이 100% 복구되어 모든 요원이 생존했습니다."
    *   **Normal / Bad Ending (가까스로 탈출)**: 오답이 기준치 이상 누적되었을 경우.
        *   *(예시)* "오라클-X: 치명적인 오류가 누적되었습니다. 메인 동력은 파괴되었지만, 보조 포드로 간신히 탈출에 성공했습니다."
*   **기대 효과**: 학생들이 문제를 대충 찍어서 맞추는 것을 방지하고, 완벽한 엔딩을 보기 위해 재도전하거나 더 신중하게 문제를 풀도록 유도합니다.

### 5. 이스터에그 및 시크릿 코드 (Easter Eggs & Secret Codes)
*   **개요**: 전체 단원이나 각 문제의 배경 및 힌트 속에 조각난 단서를 숨겨두고, 이를 조합해 숨겨진 요소를 해금하는 시스템.
*   **구현 아이디어**:
    *   각 문제의 배경 이미지 구석에 희미한 알파벳이나 숫자를 배치.
    *   학생들이 단서를 모아 게임 종료 후 특정 입력 칸에 '시크릿 코드'를 입력하면 숨겨진 쿠키 스토리 대사나 특수 칭호, 뱃지 등을 부여.
*   **기대 효과**: 문제 풀이 외에도 화면 곳곳을 관찰하는 재미를 주며, 학생들 간에 숨겨진 코드를 찾는 소통과 흥미를 유발합니다.

### 6. 글리치(Glitch) 및 UI 파괴 효과 (시각적 압박 강화)
*   **개요**: 오답이 누적될수록 경고창만 뜨는 것이 아니라, 화면 전체에 화이트 노이즈(치지직거리는 효과)가 끼거나 텍스트가 깨져서 보이는 '글리치 CSS'를 적용.
*   **구현 아이디어**: 3번 틀리기 직전에는 화면에 금이 가는 연출을 더해 극한의 긴장감을 제공.

### 7. 인벤토리 및 아이템 수집 시스템
*   **개요**: 20문제가 단순히 독립적으로 존재하는 것이 아니라, 5번 문제에서 '녹슨 열쇠', 12번 문제에서 '해독 칩' 등을 획득하는 인벤토리 UI를 우측에 추가.
*   **구현 아이디어**: 마지막 20번 문제에서 "지금까지 모은 아이템을 조합해 최종 코드를 만드세요"와 같은 방식으로 복합적인 성취감을 유도.

### 8. 실시간 생존자(리더보드) 티커 바
*   **개요**: 구글 시트(GAS) 연동의 장점을 살려, 화면 상단에 띠 전광판(Ticker)을 만들고 실시간 중계 메시지를 송출.
*   **구현 아이디어**: "현재 2반 김ㅇㅇ 요원, 15번 구역 통과!", "생존자: 12명 / 탈락자: 5명" 같은 메시지를 띄워 교실에서 진행 시 친구들과의 엄청난 경쟁심과 몰입을 유발.
