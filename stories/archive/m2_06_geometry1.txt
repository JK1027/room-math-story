# 에피소드 6. 피라미드 건축가의 비밀

## 1. [수업 개요]
- **대상 학년:** 중학 2학년
- **핵심 단원:** 도형의 성질 1 (이등변삼각형, 직각삼각형, 삼각형의 내심과 외심, 사각형의 성질)
- **플레이 타임:** 45분 (스토리 리딩 5분 + 문제 풀이 35분 + 엔딩 5분)

## 2. [교사용 오프닝 스크립트]
"고대 이집트의 위대한 건축가 임호텝이 남긴 파피루스 설계도가 발견되었습니다. 이 설계도에는 삼각형과 사각형의 기하학적 성질을 이용한 20개의 암호가 걸려 있습니다. 여러분은 임호텝의 제자가 되어 도형의 성질(내심, 외심, 평행사변형 등)을 파악하고 설계도의 봉인을 해제해야 합니다!"

## 3. [학생용 미션 지문] (학생 배포용 활동지)

### 📐 제 1구역: 완벽한 기초 (이등변삼각형과 직각삼각형)
- **Q1.** 두 변의 길이가 같은 삼각형을 무엇이라 하는가?
- **Q2.** 이등변삼각형의 두 ( ? )의 크기는 같다. 빈칸에 알맞은 말은? (힌트: 밑각)
- **Q3.** 꼭지각이 40도인 이등변삼각형의 한 밑각의 크기를 구하시오.
- **Q4.** 직각삼각형의 합동 조건 두 가지를 영어 기호로 쓰시오. (예: RHS, RHA)
- **Q5.** 빗변의 길이와 한 예각의 크기가 같은 두 직각삼각형은 서로 합동이다. 이 조건을 무엇이라 하는가?

### 🎯 제 2구역: 신성한 중심점 (삼각형의 내심과 외심)
- **Q6.** 삼각형의 세 꼭짓점을 지나는 원을 외접원이라 하고, 그 중심을 무엇이라 하는가?
- **Q7.** 둔각삼각형의 외심은 삼각형의 ( 내부 / 외부 )에 위치한다.
- **Q8.** 직각삼각형의 외심은 빗변의 ( ? )에 위치한다. 빈칸은?
- **Q9.** 삼각형의 세 내각의 이등분선이 만나는 점을 무엇이라 하는가?
- **Q10.** 삼각형의 내심에서 세 ( ? )에 이르는 거리는 같다. 빈칸은? (힌트: 변)

### 🔲 제 3구역: 대칭의 돌림판 (평행사변형의 성질)
- **Q11.** 마주 보는 두 쌍의 대변이 각각 평행한 사각형을 무엇이라 하는가?
- **Q12.** 평행사변형에서 마주 보는 대변의 길이는 서로 ( 같다 / 다르다 ).
- **Q13.** 평행사변형에서 마주 보는 두 대각의 크기는 서로 ( 같다 / 다르다 ).
- **Q14.** 평행사변형의 두 대각선은 서로 다른 것을 ( ? ) 한다. 빈칸은? (힌트: 이등분)
- **Q15.** 이웃하는 두 내각의 크기의 합이 항상 180도인 사각형은 무엇인가? (평행사변형)

### 🔶 제 4구역: 왕의 보석 (여러 가지 사각형)
- **Q16.** 네 내각의 크기가 모두 90도로 같은 사각형을 무엇이라 하는가?
- **Q17.** 직사각형의 두 대각선의 길이는 서로 ( 같다 / 다르다 ).
- **Q18.** 네 변의 길이가 모두 같은 사각형을 무엇이라 하는가?
- **Q19.** 마름모의 두 대각선은 서로 수직으로 만난다. ( O / X )
- **Q20.** 직사각형인 동시에 마름모인 사각형, 즉 네 변의 길이와 네 각의 크기가 모두 같은 사각형의 이름은?

## 4. [교사용 진행 가이드 및 힌트]
- **Q3 힌트:** $(180 - 40) \div 2 = 70$.
- **Q8 힌트:** 직각삼각형의 빗변의 중점은 외심입니다. 중요한 성질이므로 강조합니다.
