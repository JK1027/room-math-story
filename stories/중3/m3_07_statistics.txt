# 에피소드 7. 은하계 연합의 산포도 데이터베이스

## 1. [수업 개요]
- **대상 학년:** 중학 3학년
- **핵심 단원:** 통계 (대푯값과 산포도, 분산과 표준편차, 산점도와 상관관계)
- **플레이 타임:** 45분 (스토리 리딩 5분 + 문제 풀이 35분 + 엔딩 5분)

## 2. [교사용 오프닝 스크립트]
"우주 은하연합의 거대한 중앙 통계 분석실 '산포도 데이터베이스'에 도착했습니다. 이곳의 방대한 우주선 궤도 예측 자료와 기후 데이터는 통계 수치 결계로 암호화되어 있어 시스템 조작이 불가능합니다. 우주선 통계 변량들의 대푯값과 흩어진 정도를 나타내는 산포도를 계산하여 제한시간 45분 내에 시스템 락을 해제하고 무사히 궤도를 복구하여 귀환하십시오!"

## 3. [학생용 미션 지문]

### 🔮 제 1막: 대표의 수치 (대푯값 - 평균, 중앙값, 최빈값)
- **Q1.** 자료 전체의 특징을 대표적으로 나타내는 값을 통틀어 무엇이라 부르는가? (한글로 쓰시오. 예: 대푯값)
- **Q2.** 자료의 값을 모두 더하여 자료의 개수로 나눈 값을 구하시오. (두 글자 한글로 쓰시오. 예: 평균)
- **Q3.** 자료의 값을 크기순으로 나열했을 때 가장 한가운데에 위치하는 값을 무엇이라 하는가? (세 글자 한글로 쓰시오. 예: 중앙값)
- **Q4.** 자료의 값 중 가장 자주 나타나는 값을 무엇이라 하는가? (세 글자 한글로 쓰시오. 예: 최빈값)
- **Q5.** 다섯 개의 우주 변량 데이터 1, 3, 5, 7, 9 의 평균을 구하시오.
- **Q6.** 여섯 개의 우주 데이터 2, 4, 6, 8, 10, 12 의 중앙값을 구하시오.
- **Q7.** 데이터 3, 5, 5, 7, 8, 8, 8 의 최빈값을 구하시오.
- **Q8.** 대푯값 중 극단적인 값(아주 크거나 아주 작은 값)에 영향을 가장 적게 받아, 치우친 자료에 유용한 대푯값을 쓰시오. (세 글자 한글로 쓰시오. 예: 중앙값)

### 📜 제 2막: 분산의 경계 (산포도와 편차의 성질)
- **Q9.** 변량들이 평균으로부터 떨어져 있는 차이(즉, 변량 - 평균)를 무엇이라 부르는가? (두 글자 한글로 쓰시오. 예: 편차)
- **Q10.** 변량들의 편차의 총합은 항상 얼마인가? (숫자로 적으시오.)
- **Q11.** 편차의 제곱의 평균을 구하여 산포도를 나타내는 통계 값을 무엇이라 부르는가? (두 글자 한글로 쓰시오. 예: 분산)
- **Q12.** 분산의 음이 아닌 양의 제곱근 값을 구하여 실제 변량과 단위를 맞춘 산포도 값을 무엇이라 하는가? (네 글자 한글로 쓰시오. 예: 표준편차)
- **Q13.** 자료들의 흩어진 정도를 나타내는 하나의 수치를 통틀어 무엇이라 하는가? (삼항 구조의 세 글자 한글로 쓰시오. 예: 산포도)
- **Q14.** 세 변량 3, 5, 7 의 분산을 구하시오. (슬래시를 사용해 분수로 나타내시오. 예: 8/3)
- **Q15.** 네 변량 1, 3, 5, 7 의 표준편차를 구하시오. (루트 기호를 사용하여 나타내시오. 예: \sqrt{5})

### 🪄 제 3막: 상관의 흐름 (산점도와 상관관계)
- **Q16.** 두 변량의 상관관계를 한눈에 알아보기 위해 좌표평면 위에 점으로 나타낸 그림을 무엇이라 하는가? (세 글자 한글로 쓰시오. 예: 산점도)
- **Q17.** 한 변량 $x$의 값이 증가할 때, 다른 변량 $y$의 값도 대체로 증가하는 관계를 무슨 상관관계라 하는가? (칠글자 한글로 쓰시오. 예: 양의 상관관계)
- **Q18.** 한 변량 $x$의 값이 증가할 때, 다른 변량 $y$의 값은 대체로 감소하는 관계를 무슨 상관관계라 하는가? (칠글자 한글로 쓰시오. 예: 음의 상관관계)

### 🎇 제 4막: 은하 궤도 복구 (상관관계의 실생활 판별)
- **Q19.** "사람들의 키와 지능지수(IQ) 사이의 상관관계"는 어떤 관계가 있는지 쓰시오. (한글 세 글자로 쓰시오. 예: 없다)
- **Q20.** "겨울철 실외 평균 기온과 각 가정의 가스 난방비 사이의 상관관계"는 어떤 관계가 있는지 쓰시오. (칠글자 한글로 쓰시오. 예: 음의 상관관계)

## 4. [교사용 진행 가이드 및 힌트]
- **Q6 힌트:** 자료의 개수가 짝수인 경우, 가운데에 위치한 두 값의 평균을 중앙값으로 지정합니다. 즉, 6과 8의 평균인 7입니다.
- **Q14 힌트:** 평균 5를 구하고, 각 변량의 편차인 -2, 0, 2를 구합니다. 편차의 제곱은 4, 0, 4 이므로 이들의 평균인 8/3이 분산이 됩니다.

"은하 연합의 거대한 홀로그램 맵에 안정적인 우주선 궤도선들이 다시 한 번 안전한 선으로 점등됩니다. 통계 데이터 분석실의 비상 잠금장치가 모두 해제되며 우주 귀환선의 해치가 열립니다. 데이터를 마스터한 통계 분석관의 귀환을 환영합니다!"
