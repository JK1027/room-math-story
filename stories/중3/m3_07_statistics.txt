# 에피소드 7. 은하계 연합의 산포도 데이터베이스

## 1. [수업 개요]
- **대상 학년:** 중학 3학년
- **핵심 단원:** 통계 (대푯값과 산포도, 분산과 표준편차, 산점도와 상관관계)
- **플레이 타임:** 45분 (스토리 리딩 5분 + 문제 풀이 35분 + 엔딩 5분)

## 2. [교사용 오프닝 스크립트]
"우주 은하연합의 거대한 중앙 통계 분석실 '산포도 데이터베이스'에 도착했습니다. 이곳의 방대한 우주선 궤도 예측 자료와 기후 데이터는 통계 수치 결계로 암호화되어 있어 시스템 조작이 불가능합니다. 우주선 통계 변량들의 대푯값과 흩어진 정도를 나타내는 산포도를 계산하여 제한시간 45분 내에 시스템 락을 해제하고 무사히 궤도를 복구하여 귀환하십시오!"

## 3. [학생용 미션 지문]

### 🔮 제 1막: 대표의 수치 (대푯값 - 평균, 중앙값, 최빈값)
- **Q1.** 자료 전체의 특징을 대표적으로 나타내는 값을 통틀어 무엇이라 부르는가? (한글로 쓰시오. 예: 대푯값)
- **Q2.** 자료의 값을 모두 더하여 자료의 개수로 나눈 값을 구하시오. (두 글자 한글로 쓰시오. 예: 평균)
- **Q3.** 자료의 값을 크기순으로 나열했을 때 가장 한가운데에 위치하는 값을 무엇이라 하는가? (세 글자 한글로 쓰시오. 예: 중앙값)
- **Q4.** 자료의 값 중 가장 자주 나타나는 값을 무엇이라 하는가? (세 글자 한글로 쓰시오. 예: 최빈값)
- **Q5.** 다섯 개의 우주 변량 데이터 1, 3, 5, 7, 9 의 평균을 구하시오.
- **Q6.** 여섯 개의 우주 데이터 2, 4, 6, 8, 10, 12 의 중앙값을 구하시오.
- **Q7.** 데이터 3, 5, 5, 7, 8, 8, 8 의 최빈값을 구하시오.
- **Q8.** 대푯값 중 극단적인 값(아주 크거나 아주 작은 값)에 영향을 가장 적게 받아, 치우친 자료에 유용한 대푯값을 쓰시오. (세 글자 한글로 쓰시오. 예: 중앙값)

### 📜 제 2막: 분산의 경계 (산포도와 편차의 성질)
- **Q9.** 변량들이 평균으로부터 떨어져 있는 차이(즉, 변량 - 평균)를 무엇이라 부르는가? (두 글자 한글로 쓰시오. 예: 편차)
- **Q10.** 변량들의 편차의 총합은 항상 얼마인가? (숫자로 적으시오.)
- **Q11.** 편차의 제곱의 평균을 구하여 산포도를 나타내는 통계 값을 무엇이라 부르는가? (두 글자 한글로 쓰시오. 예: 분산)
- **Q12.** 분산의 음이 아닌 양의 제곱근 값을 구하여 실제 변량과 단위를 맞춘 산포도 값을 무엇이라 하는가? (네 글자 한글로 쓰시오. 예: 표준편차)
- **Q13.** 자료들의 흩어진 정도를 나타내는 하나의 수치를 통틀어 무엇이라 하는가? (삼항 구조의 세 글자 한글로 쓰시오. 예: 산포도)
- **Q14.** 세 변량 3, 5, 7 의 분산을 구하시오. (슬래시를 사용해 분수로 나타내시오. 예: 8/3)
- **Q15.** 네 변량 1, 3, 5, 7 의 표준편차를 구하시오. (루트 기호를 사용하여 나타내시오. 예: √5)

### 🪄 제 3막: 상관의 흐름 (산점도와 상관관계)
- **Q16.** 두 변량의 상관관계를 한눈에 알아보기 위해 좌표평면 위에 점으로 나타낸 그림을 무엇이라 하는가? (세 글자 한글로 쓰시오. 예: 산점도)
- **Q17.** 한 변량 x의 값이 증가할 때, 다른 변량 y의 값도 대체로 증가하는 관계를 무슨 상관관계라 하는가? (칠글자 한글로 쓰시오. 예: 양의 상관관계)
- **Q18.** 한 변량 x의 값이 증가할 때, 다른 변량 y의 값은 대체로 감소하는 관계를 무슨 상관관계라 하는가? (칠글자 한글로 쓰시오. 예: 음의 상관관계)

### 🎇 제 4막: 은하 궤도 복구 (상관관계의 실생활 판별)
- **Q19.** "사람들의 키와 지능지수(IQ) 사이의 상관관계"는 어떤 관계가 있는지 쓰시오. (한글 세 글자로 쓰시오. 예: 없다)
- **Q20.** "겨울철 실외 평균 기온과 각 가정의 가스 난방비 사이의 상관관계"는 어떤 관계가 있는지 쓰시오. (칠글자 한글로 쓰시오. 예: 음의 상관관계)

## 4. [교사용 진행 가이드 및 힌트]
- **Q1 힌트:** 자료 전체의 특징을 대표적으로 표현하는 통계적인 용어인 세 글자 '대푯값'을 떠올립니다.
- **Q2 힌트:** 모든 수치 자료의 값들을 더해서 전체 개수로 나눈 기본 가치(평균)를 한글 두 글자로 대답하게 합니다.
- **Q3 힌트:** 자료를 크기 순서대로 줄 세웠을 때 정중앙에 위치하는 값인 세 글자 '중앙값'을 구하게 합니다.
- **Q4 힌트:** 전체 변량 수치 중 가장 빈번하게 나타나는 값인 세 글자 '최빈값'을 떠올리게 유도합니다.
- **Q5 힌트:** 다섯 개의 자료를 모두 더한 값(1+3+5+7+9 = 25)을 전체 개수인 5로 나누어 줍니다.
- **Q6 힌트:** 자료의 개수가 짝수(6개)이므로, 크기순으로 배열했을 때 가운데 위치하는 두 값인 6과 8의 평균(7)을 계산합니다.
- **Q7 힌트:** 데이터 중 가장 많이 중복되어 빈도가 높은 수치(8이 3번으로 최빈)를 찾습니다.
- **Q8 힌트:** 극단치(아주 큰 수치)가 섞여 있을 때 평균은 왜곡되기 쉬우므로, 크기순으로 줄을 세운 중앙값을 대푯값으로 쓰는 것이 가장 합리적입니다.
- **Q9 힌트:** 편차를 구하는 수학 공식인 (변량 - 평균)을 상기시킵니다.
- **Q10 힌트:** 모든 변량들의 편차들을 더하면 양의 편차와 음의 편차가 완전히 상쇄되어 합은 언제나 0 이 됨을 설명합니다.
- **Q11 힌트:** 분산의 정의인 (편차의 제곱들의 평균)을 지칭하는 두 글자 단어를 쓰게 합니다.
- **Q12 힌트:** 표준편차 = √분산 관계를 만족하며 단위가 원본 변량과 동일해지는 4글자 용어입니다.
- **Q13 힌트:** 자료가 평균을 중심으로 얼마나 넓고 좁게 분포해 있는지를 총칭하는 3글자 통계 용어(산포도)를 쓰게 합니다.
- **Q14 힌트:** 평균 5를 구하고, 각 변량의 편차인 -2, 0, 2를 계산합니다. 편차제곱 4, 0, 4의 평균인 8/3이 분산이 됩니다.
- **Q15 힌트:** 자료의 분산이 5이므로, 표준편차는 이의 양의 제곱근인 √5 가 됩니다.
- **Q16 힌트:** 두 변량 사이의 상관관계를 시각적으로 한눈에 파악하기 위해 점들로 나타낸 그림인 '산점도'를 답하게 합니다.
- **Q17 힌트:** x가 증가할 때 y도 우상향하는 선형 관계를 나타내는 7글자 용어(양의 상관관계)를 쓰게 합니다.
- **Q18 힌트:** x가 증가할 때 y는 우하향하여 감소하는 경향을 보이는 7글자 용어(음의 상관관계)를 쓰게 합니다.
- **Q19 힌트:** 사람의 키 크기와 지능(IQ) 수준은 통계적으로 전혀 연관되어 있지 않으므로 상관관계가 '없다'라고 답합니다.
- **Q20 힌트:** 추운 겨울날 기온이 올라가면 따뜻해져 난방용 가스 사용량(난방비)은 떨어지므로, 음의 상관관계가 나타납니다.
