## 3. [학생용 미션 지문]

### 🔮 제 1막: 방정식의 판별 (이차방정식의 정의와 해)
- **Q1.** 다음 보기 중 이차방정식인 것의 번호를 쓰시오.
(1) 2x - 5 = 0
(2) x² - 3x + 2 = 0
(3) x³ + x² - 1 = 0
(4) x² - (x² - 2x) = 0
- **Q2.** 이차방정식 x² - 4 = 0 의 두 근을 구하시오. (± 기호를 사용하여 구하시오. 예: ±2)
- **Q3.** x=2 가 이차방정식 x² - ax + 6 = 0 의 한 근일 때, 상수 a의 값을 구하시오.
- **Q4.** 이차방정식 (x-3)(x+2) = 0 의 두 근을 구하시오. (작은 수부터 쉼표로 구분하여 적으시오. 예: -2, 3)
- **Q5.** 이차방정식 x² - 5x + 6 = 0 의 두 근을 구하시오. (작은 수부터 적으시오. 예: 2, 3)

### 📜 제 2막: 제곱의 힘 (중근과 완전제곱식의 활용)
- **Q6.** 이차방정식 x² - 8x + 16 = 0 의 근을 구하시오.
- **Q7.** 이차방정식 x² + ax + 9 = 0 이 중근을 가질 때, 양수 a의 값을 구하시오.
- **Q8.** 이차방정식 (x-2)² = 5 의 근을 구하시오. (기호 ±와 루트를 사용하여 나타내시오. 예: 2±√5)
- **Q9.** 이차방정식 x² - 6x + 2 = 0 을 (x-p)² = q 꼴로 고칠 때, 두 상수 p, q에 대하여 p+q의 값을 구하시오.
- **Q10.** 근의 공식을 사용하여 이차방정식 x² - 3x + 1 = 0 의 근을 구할 때, 분모에 들어갈 값을 구하시오.

### 🪄 제 3막: 근의 열쇠 (근의 공식과 풀이)
- **Q11.** 이차방정식 x² - 4x - 1 = 0 의 근을 구하시오. (근의 공식을 활용하여 나타내시오. 예: 2±√5)
- **Q12.** 이차방정식 2x² + 5x - 3 = 0 의 두 근을 구하시오. (작은 수부터 적으시오. 분수는 슬래시로 표현하시오. 예: -3, 1/2)
- **Q13.** 이차방정식 (x-1)(x-2) = 6 의 두 근을 구하시오. (작은 수부터 적으시오. 예: -1, 4)
- **Q14.** 두 근이 -2와 5 이고 x²의 계수가 1인 이차방정식을 구하시오. (예: x²-3x-10=0)
- **Q15.** 이차방정식 x² + 4x - 5 = 0 의 두 근 중 양수인 근을 구하시오.

### 🎇 제 4막: 톱니바퀴 재가동 (이차방정식의 활용)
- **Q16.** 이차방정식 x² - 2x = 0 의 두 근을 구하시오. (작은 수부터 적으시오. 예: 0, 2)
- **Q17.** 이차방정식 3x² - 7x + 2 = 0 의 두 근 중 정수인 근을 구하시오.
- **Q18.** 이차방정식 x² - 6x + 9 = 0 의 중근을 구하시오.
- **Q19.** 연속하는 두 자연수의 곱이 56일 때, 두 자연수 중 큰 수를 구하시오.
- **Q20.** 지면에서 초속 30m로 똑바로 쏘아 올린 물체의 t초 후의 높이가 30t - 5t² (m)이다. 이 물체의 높이가 40m가 되는 시점은 몇 초 후인지 모두 구하시오. (오름차순으로 적으시오. 예: 2, 4)

## 4. [교사용 진행 가이드 및 힌트]
- **Q9 힌트:** 좌변을 완전제곱식으로 만들기 위해 상수항을 우변으로 이항하고, 양변에 x의 계수의 반의 제곱인 9를 더해줍니다. x² - 6x + 9 = -2 + 9 \rightarrow (x-3)² = 5.
- **Q20 힌트:** 높이 식이 40이 되는 방정식을 세웁니다. 30t - 5t² = 40 \rightarrow 5t² - 30t + 40 = 0 \rightarrow t² - 6t + 8 = 0 \rightarrow (t-2)(t-4)=0.
