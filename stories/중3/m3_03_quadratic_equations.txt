# 에피소드 3. 시간의 이차방정식 톱니바퀴 탑

## 1. [수업 개요]
- **대상 학년:** 중학 3학년
- **핵심 단원:** 이차방정식 (이차방정식의 풀이, 근의 공식 및 활용)
- **플레이 타임:** 45분 (스토리 리딩 5분 + 문제 풀이 35분 + 엔딩 5분)

## 2. [교사용 오프닝 스크립트]
"멈춰버린 거대한 '시간의 탑'의 제어실에 들어섰습니다. 탑의 거대한 톱니바퀴들은 '이차방정식'의 동력 에너지 수식 결계로 고정되어 움직이지 않습니다. 톱니바퀴들의 맞물림 관계와 회전 주기를 나타내는 이차방정식의 근들을 정확하게 찾아내어 제한시간 45분 내에 시계탑을 다시 가동시키고 시간의 봉인에서 탈출하십시오!"

## 3. [학생용 미션 지문]

### 🔮 제 1막: 방정식의 판별 (이차방정식의 정의와 해)
- **Q1.** 다음 보기 중 이차방정식인 것의 번호를 쓰시오.
(1) 2x - 5 = 0
(2) x² - 3x + 2 = 0
(3) x³ + x² - 1 = 0
(4) x² - (x² - 2x) = 0
- **Q2.** 이차방정식 x² - 4 = 0 의 두 근을 구하시오. (± 기호를 사용하여 구하시오. 예: ±2)
- **Q3.** x=2 가 이차방정식 x² - ax + 6 = 0 의 한 근일 때, 상수 a의 값을 구하시오.
- **Q4.** 이차방정식 (x-3)(x+2) = 0 의 두 근을 구하시오. (작은 수부터 쉼표로 구분하여 적으시오. 예: -2, 3)
- **Q5.** 이차방정식 x² - 5x + 6 = 0 의 두 근을 구하시오. (작은 수부터 적으시오. 예: 2, 3)

### 📜 제 2막: 제곱의 힘 (중근과 완전제곱식의 활용)
- **Q6.** 이차방정식 x² - 8x + 16 = 0 의 근을 구하시오.
- **Q7.** 이차방정식 x² + ax + 9 = 0 이 중근을 가질 때, 양수 a의 값을 구하시오.
- **Q8.** 이차방정식 (x-2)² = 5 의 근을 구하시오. (기호 ±와 루트를 사용하여 나타내시오. 예: 2±√5)
- **Q9.** 이차방정식 x² - 6x + 2 = 0 을 (x-p)² = q 꼴로 고칠 때, 두 상수 p, q에 대하여 p+q의 값을 구하시오.
- **Q10.** 근의 공식을 사용하여 이차방정식 x² - 3x + 1 = 0 의 근을 구할 때, 분모에 들어갈 값을 구하시오.

### 🪄 제 3막: 근의 열쇠 (근의 공식과 풀이)
- **Q11.** 이차방정식 x² - 4x - 1 = 0 의 근을 구하시오. (근의 공식을 활용하여 나타내시오. 예: 2±√5)
- **Q12.** 이차방정식 2x² + 5x - 3 = 0 의 두 근을 구하시오. (작은 수부터 적으시오. 분수는 슬래시로 표현하시오. 예: -3, 1/2)
- **Q13.** 이차방정식 (x-1)(x-2) = 6 의 두 근을 구하시오. (작은 수부터 적으시오. 예: -1, 4)
- **Q14.** 두 근이 -2와 5 이고 x²의 계수가 1인 이차방정식을 구하시오. (예: x²-3x-10=0)
- **Q15.** 이차방정식 x² + 4x - 5 = 0 의 두 근 중 양수인 근을 구하시오.

### 🎇 제 4막: 톱니바퀴 재가동 (이차방정식의 활용)
- **Q16.** 이차방정식 x² - 2x = 0 의 두 근을 구하시오. (작은 수부터 적으시오. 예: 0, 2)
- **Q17.** 이차방정식 3x² - 7x + 2 = 0 의 두 근 중 정수인 근을 구하시오.
- **Q18.** 이차방정식 x² - 6x + 9 = 0 의 중근을 구하시오.
- **Q19.** 연속하는 두 자연수의 곱이 56일 때, 두 자연수 중 큰 수를 구하시오.
- **Q20.** 지면에서 초속 30m로 똑바로 쏘아 올린 물체의 t초 후의 높이가 30t - 5t² (m)이다. 이 물체의 높이가 40m가 되는 시점은 몇 초 후인지 모두 구하시오. (오름차순으로 적으시오. 예: 2, 4)

## 4. [교사용 진행 가이드 및 힌트]
- **Q1 힌트:** 주어진 등식을 모두 한 변(좌변)으로 이항하여 정리했을 때, (x에 대한 2차식) = 0 꼴이 되는 식을 찾게 합니다.
- **Q2 힌트:** x² = 4 로 이항한 뒤 제곱근의 정의를 활용해 x = ±√4 를 유도하여 구하게 합니다.
- **Q3 힌트:** 방정식의 근이 2이므로 식의 x 자리에 2를 대입하면 등식이 성립한다는 것을 이용하여 a를 구합니다.
- **Q4 힌트:** 두 식의 곱이 0이 되려면 x-3 = 0 또는 x+2 = 0 이 되어야 함을 지도합니다.
- **Q5 힌트:** 좌변을 인수분해하여 (x-2)(x-3) = 0 으로 변형한 뒤 두 근을 구하게 유도합니다.
- **Q6 힌트:** 좌변이 완전제곱식 (x-4)² = 0 으로 인수분해되므로 중근 4를 얻게 됩니다.
- **Q7 힌트:** 이차방정식이 중근을 가지려면 상수항인 9가 x의 계수인 a의 반의 제곱인 (a/2)²과 같아야 합니다.
- **Q8 힌트:** 제곱근을 이용하여 x-2 = ±√5 가 됨을 파악하고 x에 대해 정리합니다.
- **Q9 힌트:** 완전제곱식 유도를 위해 상수항을 우변으로 이항하여 x²-6x = -2 로 만들고, 양변에 x 계수의 반의 제곱인 9를 더해 정리합니다.
- **Q10 힌트:** 근의 공식 x = (-b ± √(b² - 4ac)) / 2a 에서 분모는 항상 2a가 됨을 활용합니다.
- **Q11 힌트:** 근의 공식을 적용합니다. 짝수 공식 x = (-b' ± √(b'² - ac)) / a 를 사용하면 더욱 간단하게 구할 수 있습니다.
- **Q12 힌트:** 대각선 곱셈법을 이용하여 좌변을 (2x-1)(x+3) = 0 으로 인수분해하여 두 근을 구합니다.
- **Q13 힌트:** 괄호를 먼저 전개하여 x²-3x+2 = 6 으로 만든 후, 우변의 6을 이항해 x²-3x-4 = 0 형태로 정리한 다음 풀이합니다.
- **Q14 힌트:** 두 근이 a, b일 때 이차방정식은 (x-a)(x-b) = 0 임을 이용해 (x+2)(x-5) = 0 을 전개합니다.
- **Q15 힌트:** 좌변을 (x+5)(x-1) = 0 으로 인수분해한 후 얻어지는 두 근 -5와 1 중 0보다 큰 양수 근을 찾습니다.
- **Q16 힌트:** 공통인수인 x를 묶어내어 x(x-2) = 0 꼴로 변형한 뒤 해를 구합니다.
- **Q17 힌트:** 좌변을 (3x-1)(x-2) = 0 으로 인수분해한 후 얻어지는 두 근 1/3과 2 중 정수인 근을 고릅니다.
- **Q18 힌트:** 좌변이 (x-3)² = 0 완전제곱식이 되므로 중복되는 해(중근)는 3입니다.
- **Q19 힌트:** 작은 자연수를 x라 두면 큰 자연수는 x+1 이며, x(x+1) = 56 식을 세워 이차방정식을 풀고 큰 수인 x+1을 구합니다.
- **Q20 힌트:** 높이 식 30t - 5t² 이 40이 되는 방정식을 수립합니다. 양변을 -5로 나누어 t² - 6t + 8 = 0 으로 변형해 해를 구합니다.
